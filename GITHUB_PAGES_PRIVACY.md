# GitHub Pages Privacy Policy Setup

Use this when Google Play asks for the privacy policy URL.

## Files Ready To Publish

- `docs/index.html`
- `docs/privacy-policy/index.html`

## GitHub Steps

1. Go to https://github.com/new
2. Create a public repository named:

```text
bcn-plant-scout
```

3. Upload the project files or at least the `docs` folder.
4. Go to the repository settings.
5. Open `Pages`.
6. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
7. Save.

## Privacy Policy URL

After GitHub Pages finishes publishing, the URL should be:

```text
https://greenjosh327.github.io/bcn-plant-scout/privacy-policy/
```

If the GitHub username or repository name is different, adjust the URL:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPO-NAME/privacy-policy/
```

Paste that public `https://` URL into Google Play Console.
