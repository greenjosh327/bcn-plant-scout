## App Name

BCN Plant Scout

## Short Description

Plant photos, GPS field records, return notes, exports, and BCN nursery sync.

## Full Description

BCN Plant Scout is a local-first field notebook for plant scouting, seed-source tracking, nursery notes, and return visits.

Built for Base Camp North, the app helps record plant observations with photos, GPS coordinates, plant identification suggestions, common and scientific names, collection interests, return dates, and field notes.

Use it to:

- Take or select plant photos.
- Capture GPS points and location accuracy.
- Get plant ID suggestions from photos.
- Save alternate common names.
- Track seeds, cuttings, nuts, fruit, scion wood, or observation-only records.
- Mark plants as ready now, return later, not ready, collected, or do not collect.
- Add return dates and local reminder heads-up settings.
- Add extra photos to existing records.
- Open saved plants in Google Maps.
- Export CSV, GeoJSON, or ZIP packages for GIS workflows.
- Sync records and photos to Supabase when signed in.

BCN Plant Scout is designed for field work first: save records locally, then export or sync when ready.

## Data Safety Notes

- Email address for sign-in.
- Photos selected or taken by the user.
- Precise location for saved plant records.
- User-entered plant notes and collection information.

Data use:

- App functionality.
- Account management.
- Cloud sync when signed in.
- Plant identification.

Data sharing:

- Photos may be sent to Pl@ntNet for plant identification.
- Data may be stored in Supabase when the user signs in and uploads records.
- Google Maps is opened externally when the user taps map links.
